# Story Agent Workbench

## Goal
做一个面向长篇游戏剧本/叙事资料的通用 AI 工作台，不只服务单个剧本。

## Core use cases
- 阅读已有设定、场景、草稿
- 回答“全局上看”的问题
- 追人物关系、阵营链条、伏笔回收网
- 支持普通 RAG 和 GraphRAG
- 用一个主代理把问题路由到文本检索或图谱检索
- 后续接聊天界面

## First version
- Python 项目骨架
- 文档导入
- 普通 RAG 最小版
- Graph 层接口占位
- 清晰的目录结构
- 易读、易改、不过度设计

## Not now
- 漂亮前端
- 多人协作系统
- 复杂数据库
- 生产部署

## Developer profile
- GitHub 新手
- 只会一点 Python 和一点 C
- 希望代码简单、注释清楚、结构稳定
